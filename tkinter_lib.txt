Интерфейс программы состоит из меню программы из трех пунктов ("Построить график", "Справка", "Выход")
и поля для отрисовки графика функции.
При нажатии на пункт "Построить график" пользователю предлагается выбрать одну из шести возможных функций, после чего выбрать 
начальное и конечное значение Х. Сразу после этого произойдет отрисовка графика выбранной функции. 
При нажатии на пункт "Справка" открывается данный текстовый файл со справкой о программе.
При нажатии на пункт "Выход" происходит выход из программы.

При выборе функции в меню построения графика с помощью функции clear() происходит очищение поля для отрисовки графика, после чего
с помощью функции choose_X открывается выбор диапазона значений. В случае некорректного ввода (x1 >= x2) появляется сообщение об 
ошибке. При корректном вводе происходит отрисовка одной из выбранных функций (название функции передается в виде lambda функции в 
меню графика).
